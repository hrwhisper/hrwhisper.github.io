感知机

感知机（Perceptron）是一个二分类的线性模型。它于1957年被提出，是支持向量机和神经网络的基础。

感知机模型

模型输入数据为：{(x1,y1),(x2,y2),⋯ ,(xn,yn)}\{({\bf x_1}, y_1),({\bf x_2}, y_2),\cdots,({\bf x_n}, y_n) \}{(x1​,y1​),(x2​,y2​),⋯,(xn​,yn​)}，其中xix_ixi​为样本 i 的特征向量，yi∈{+1,−1}y_i\in \{+1,-1\}yi​∈{+1,−1}为类别标签。

输入到输出空间的映射为：

f(x)=sign(wTx+b)f(x) = {\rm sign}({\bf w^Tx}+b) f(x)=sign(wTx+b)

上述的sign为符号函数，其值如下：

sign(t)={+1,t≥0−1,t<0{\rm sign}(t) = \begin{cases} +1 , & t\ge 0 \\ -1, & t\lt 0 \end{cases} sign(t)={+1,−1,​t≥0t<0​

感知机的几何解释

感知机的目标是求一个超平面使得特征空间划分为两部分，从而实现分类的效果。

线性方程 wTx+b=0{\bf w^T x}+b = 0wTx+b=0，w为超平面的法向量，b是超平面的截距。

w为啥是法向量？证明如下：

设x0,x1{\bf x_0,x_1}x0​,x1​是平面上的任意两个点，则显然有：

\begin{align} {\bf w^T x_0} + b = 0\\ {\bf w^T x_1} + b = 0\\ \end{align}

两式相减得：

\begin{align} {\bf w^T (x_0 - x_1)} = 0\\ \end{align}

而x0−x1\bf x_0 - x_1x0​−x1​是平面上的向量，而x0−x1\bf x_0 - x_1x0​−x1​与w内积为0，因此，w就是法向量（垂直于平面内所有向量）。

学习策略

那么，如何训练出感知机呢？

我们需要有一个损失函数，这样我们才能不断的优化它，最后达到一个较为理想的值。

既然是分类问题，一个自然的选择是误分类点的总数（即0-1损失函数）

l0−1={0f(xi)=yi1f(xi)≠yil_{0-1}= \begin{cases} 0&f({\bf x_i}) = y_i\\ 1& f({\bf x_i}) \ne y_i \end{cases} l0−1​={01​f(xi​)=yi​f(xi​)​=yi​​

但是这个损失函数不是参数w,b的连续可导函数，不容易进行优化。

因此，感知机采用的是误分类点到超平面S的总距离。

设点x0\bf x_0x0​到超平面 wTx0+b=0{\bf w^T x_0}+b = 0wTx0​+b=0的距离为d。x1\bf x_1x1​为平面上的任意一点，则∣ ⁣∣x0−x1∣ ⁣∣|\!|{\bf x_0} - {\bf x_1}|\!|∣∣x0​−x1​∣∣投影到w\bf ww的距离为：

\begin{align} \rm proj_w({\bf x_0-x_1}) &=|\!|{\bf x_0-x_1}|\!|\cdot |\cos({\bf w,x_0-x_1})| \\&=|\!|{\bf x_0-x_1}|\!|\frac{|{\bf w^T(\bf x_0 - x_1)}|}{ {|\!|{\bf w}|\!|\ |\!|\bf x_0-x_1|\!|} } \\&=\frac{|{\bf w^T(\bf x_0 - x_1)}|}{ {|\!|{\bf w}|\!|} } \\&=\frac{|{\bf w^T\bf x_0 -w^Tx_1}|}{ {|\!|{\bf w}|\!|} } \\&=\frac{|{\bf w^T\bf x_0} +b|}{ {|\!|{\bf w}|\!|} } \tag{2-1} \end{align}

式2-1即为点x0{\bf x_0}x0​到超平面S的距离。

对于分类错误的样本(xi,yi)({\bf x_i}, y_i)(xi​,yi​)有：

yi(wTxi+b)<0y_i({\bf w^T x_i}+ b) \lt 0 yi​(wTxi​+b)<0

因此，误分类点到平面的距离为：

−1∣ ⁣∣w∣ ⁣∣yi(wTxi+b）-\frac{1}{|\!|{\bf w}|\!|}y_i({\bf w^T x_i}+ b） −∣∣w∣∣1​yi​(wTxi​+b）

考虑所有的分类错误的点集合M，则所有误分类点到超平面S的总距离：

−1∣ ⁣∣w∣ ⁣∣∑xi∈Myi(wTxi+b）-\frac{1}{|\!|{\bf w}|\!|}\sum_{x_i\in M}y_i({\bf w^T x_i} + b） −∣∣w∣∣1​xi​∈M∑​yi​(wTxi​+b）

不考虑1∣∣w∣∣\frac{1}{||{\bf w}||}∣∣w∣∣1​，就得到感知机的损失函数：

\begin{align*} L(w, b) &= -\sum_{x_i\in M}y_i({\bf w^T x_i}+ b), i = 1,2...,n\\ &= \sum_{i=1}^n\max(0, -y_i({\bf w^T x_i}+ b)) \end{align*}\tag{2-2}

损失函数L(w,b)L({\bf w}, b)L(w,b) 是非负的，若没有误分类点，则L(w,b)=0L({\bf w}, b)=0L(w,b)=0

PS: 为什么能不考虑1∣∣w∣∣\frac{1}{||{\bf w}||}∣∣w∣∣1​呢？李航老师将yi(wTxi+b)y_i({\bf w^T x_i} + b)yi​(wTxi​+b)称为函数间隔，而称1∣∣w∣∣yi(wTxi+b)\frac{1}{||{\bf w}||}y_i({\bf w^T x_i} + b)∣∣w∣∣1​yi​(wTxi​+b)为几何间隔。感知机省略了1∣∣w∣∣\frac{1}{||{\bf w}||}∣∣w∣∣1​相当于是最小化函数间隔。后面会讲到SVM则是实实在在的几何间隔。几何间隔反应了点到超平面的垂直距离，而函数间隔则可以表示正确性以及分错的置信度。SVM想要鲁棒性(robust)强，因此采用最大化几何间隔。而感知机只想把所有的点都分对，只需要函数间隔即可。

梯度下降求解

由于L(w, b) 是w,b的连续可导函数，我们可以用随机梯度下降法进行求解。

\begin{align*} \frac{\partial L({\bf w}, b)}{\partial {\bf w} } &= -\sum_{x_i\in M}y_i {\bf x_i}\\ \frac{\partial L({\bf w}, b)}{\partial b} &= -\sum_{x_i\in M}y_i \end{align*}

因此，感知机学习算法（Perceptron Learning Algorithm, PLA）如下(随机梯度下降)：

  1. 给定学习率η\etaη，随机选取初值w, b

  2. 选取分类错误的数据 xi,yi{\bf x_i}, y_ixi​,yi​按下面的式子更新，直到没有错误为止

w←w+ηyixib←b+ηyi(2-3) \begin{aligned} {\bf w} \leftarrow& {\bf w} + \eta y_i{\bf x_i}\\ b \leftarrow& b + \eta y_i \end{aligned}\tag{2-3} w←b←​w+ηyi​xi​b+ηyi​​(2-3)

感知机选取不同的初值或选取不同的分类点，解可能不同。为了得到唯一的超平面，需要对分离超平面增加约束条件。（其实就是SVM的想法）

当数据集 线性可分 时（即存在一个超平面，能将数据集中正例负例正确的划分到超平面两侧），可以证明，感知机一定会收敛（即没有错误），在本小节最后给出相应的证明。

当线性不可分的时候，往往设定迭代次数。

一种几何解释

在台大的林轩田《机器学习基石》课程中，提到了这种修正方法：

对于分类错误的样本，有（把b放入了w，对应的x为1就得到）：

sign(wTxi)≠yi{\rm sign}({\bf w^T x_i}) \not =y_{i} sign(wTxi​)​=yi​

有两种情况：

  • y是+1，但是模型判断出来是负值。也就是说此时w与x之间的角度太大，因此需要把w往靠近x的方向旋转使它们的角度变小。可以通过w←w+yixi{\bf w} \leftarrow {\bf w} + y_i{\bf x_i}w←w+yi​xi​达到这个目的

  • y是-1，但是模型判断出来是正值。也就是说此时w与x之间的角度太小，因此需要把w往远离x的方向旋转使它们的角度变大。考虑到负号，其实也可以可以通过w←w+yixi{\bf w} \leftarrow {\bf w} + y_i{\bf x_i}w←w+yi​xi​达到这个目的

感知机的一些变形

前面提到了感知机的学习算法PLA，其可以容易的扩展到在线学习、批量学习、以及核函数形式。

在线学习 Online learning

Online learning refers to the learning mode in which the model update is performed each time a single observation is received

在线学习即使用新观测到一条数据来更新模型。

回顾一下PLA，PLA是遍历数据，选一条分错的数据按2-3式进行更新。其实就是随机梯度下降法。这很容易扩展到在线学习中。给定一条新的数据，只要数据分错，那么继续按2-3式更新。

  1. 给定学习率η\etaη，随机选取初值w

  2. 给定数据(xi,yi)({\bf x_i},y_i)(xi​,yi​)，若yi≠sign(wTxi)y_i \ne sign({\bf w^Tx_i})yi​​=sign(wTxi​)按下面的式子更新

w←w+ηyixi{\bf w} \leftarrow {\bf w} + \eta y_i{\bf x_i} w←w+ηyi​xi​

核化感知机

如果你不知道核函数是什么，可以先看完我的SVM系列（尤其是深入理解SVM(二) – 核函数和软边距）后再来看该内容。

为了使用核方法，首先要将w\bf ww用xi\bf x_ixi​线性表示。根据假设其初值为0，根据PLA算法，可以看出

w=∑i=1nαiyixi{\bf w} = \sum_{i=1}^n\alpha_iy_i{\bf x_i} w=i=1∑n​αi​yi​xi​

其中，αi=niη\alpha_i= n_i\etaαi​=ni​η表示样本xix_ixi​被误分类的次数nin_ini​乘以η\etaη（每误分一次就要使用w←w+ηyixi{\bf w} \leftarrow {\bf w} + \eta y_i{\bf x_i}w←w+ηyi​xi​更新一次）

因此，感知机可以写为

\begin{align*} f({\bf x}) &= \rm sign({\bf w^Tx})\\ &= \rm sign(\sum_{i=1}^n\alpha_i{\bf y_i{\bf x_i}^Tx})\\ & = \rm sign(\sum_{i=1}^n\alpha_iK({\bf x_i, x})) \end{align*}

其学习方式为:

  1. 给定学习率η\etaη，αi=0\alpha_i = 0αi​=0

  2. 对于每个训练数据(xj,yj)({\bf x_j},y_j)(xj​,yj​)，若yi≠sign(∑i=1nαiK(xi,xj))y_i \ne sign(\sum_{i=1}^n\alpha_iK({\bf x_i, x_j}))yi​​=sign(∑i=1n​αi​K(xi​,xj​)) 按下面的式子更新，直到达到迭代次数或满足终止条件：

αi←αi+η \alpha_i \leftarrow \alpha_i + \eta αi​←αi​+η

  • 感知机寻找超平面wTx+b=0{\bf w^T x}+b = 0wTx+b=0 来划分数据，采用sign(wTx+b)\rm sign({\bf w^T x}+b)sign(wTx+b)来判别正负类
