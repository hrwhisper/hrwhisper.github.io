使用google map API获取经纬度

发表于 2017-02-08 分类于 code ， study ， OtherCodes ， WEB前端 阅读次数： 评论数：

当年在做毕设的时候一个功能是在地图上点击，然后获取对应的地区的经纬度，然后用来追踪该地区发布的tweets

效果

  • 鼠标左键点击在地图上增加标记，并显示经纬度信息
  • 鼠标右键取消地图标记
  • 关闭模态框后，地图上所有标记点的经纬度信息都会出现在文本框中

首先让大家看看效果 点击获取经纬度坐标
